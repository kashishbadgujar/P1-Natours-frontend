#  Natours Project

A responsive and modern **Tour booking landing page** built using **HTML, CSS, and SCSS**, focusing on advanced UI design, animations, and layout techniques.

---

## 🚀 Features

- Responsive design & CSS animations
- Flexbox & Grid layouts
- SCSS architecture
- BEM naming methodology
- Card flip effects
- Popup modal functionality

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- SCSS (Sass)
---

## 📂 Project Structure

```
project/
 ├── index.html
 ├── css/
 │    └── style.css
 ├── sass/
 ├── imgs/
 └── README.md
```

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/kashishbadgujar/P1-Natours-frontend.git
```

2. Navigate to the project folder:

```
cd P1-Natours-frontend
```

3. Install dependencies:

```
npm install
```

4. Compile SCSS to CSS (if using Sass):

```
npm run compile:sass
```

5. Open `index.html` in your browser
